# Development

The base server profiles for each Ping app supported on the Ping Cloud platform are maintained and versioned through this repository: 
[ping-cloud-profiles](https://gitlab.corp.pingidentity.com/ping-cloud-private-tenant/ping-cloud-docker/-/blob/master/ping-cloud-profiles)

Any customizations made here will override the base profiles. Use the above location as a reference for how to structure the profiles directory.
